### Criando um novo branch pelo terminal

Comando necessário para criação de uma nova branch via terminal:


git checkout -b (nome da nova branch)

Este comando vai criar e trocar para essa nova branch

Para criar a brach pelo github

1 - Acessar o repositório
2 - Ir na sessão de listagens das branchs
3 - Inserir o nome da nova branchs
4 - Confirmar a criação dessa nova branch

Caso queira excluir uma branch pelo github
1 - Vá na sessão de listagens de branchs
2 - Clique no ícone de lixeira para realizar a exclusão dessa branch
